Tools for Tracking Machine Learning Experiments

  • 実験管理ツール
    • artemis: Artemis aims to get rid of all the boring involved in machine learning.
    • scared: Help you configure, organize, log and reporduce experiments.
    • MLflow: Open source platform for the machine learning lifesycle.
  • ハイパーパラメータ最適化
    • hyperopt: Distributed asynchronous hyperparameter optimization in python.
    • SMAC v3: Sequential model-based algorithm configuration.
  • ハイパーパラメータ最適化 + 実験管理ツール
    • optuna: A hyperparameter optimization frameowrk.
    • randopt: Streamlined machine learning experiment management.
    • test-tube: Python library to easily log experiments and paralleize hyperparameter search for neural networks.
  • 実験結果可視化ツール
    • ChainerUI: User interface for chainer.
      • 2019.12.5 にこれ以上 Chainer のメジャーバージョンアップはしないことが発表されている。
    • TensorBoard: TensorFlow’s Visualization toolkit.
    • Visdom: A flexible tool for creating, organizing, and sharing visualization of live, rich data.
