Billion-scale semi-supervised learning for image classification

2020-02-28 14:50:25 +0000 UTC 2020-03-29 06:04:28 +0000 UTC
  • arXiv
  • 著者: I. Zeki Yalniz, Hervé Jégou, Kan Chen, Manohar Paluri, Dhruv Mahajan
  • 投稿日: 2019.3.2

ラベル付きのデータセットとラベル無しの大量のデータセットを使って蒸留を行う手法の提案です。

  1. 教師モデルをラベル付きデータセットで学習
  2. 教師モデルでラベルなしの大量データセットにラベルをつけ、信頼度をスコアリング
  3. スコアリング結果に基づき各クラスからデータをを取得して新しいデータセットを作成
  4. 新しいデータセットで生徒モデルの事前学習を実施
  5. 生徒モデルの Fine Tuning をラベル付きデータセットで実施
