Kaggle API

  • GitHub Kaggle/kaggle-api
    • Kernel Metadata

Kaggle 公式のコマンドで操作するための API です。 Python で実装されています。 ターミナルから利用するこまんどですが、 pip でインストールするパッケージのため、 普通に python スクリプト中からコマンドを呼び出すことができます。

Tips

  • google colaboratory で kaggle api を利用する場合は、 /root/.kaggle の下に kaggle.json を設置する必要がある。

  • 2019.12.24 Qiita Kaggle でのソースコード管理の手助け
    • Kaggle Notebook に提出するコードを通常の git 管理環境から変更少なく実現する方法が書かれいます。
    • 概要としては、通常通り複数ファイルに分割したファイルを全て base64 エンコードして提出用のファイルに埋め込みます。 その後、埋め込んだ提出用ファイルをコピペして Notebook に張り付けることで、 分割ファイルを全てパッケージとしてインストールするようです。
    • いつも通り自分の好きな環境で記述してから実行できるので非常に良い方法だと思います。
    • デメリットは、提出したファイルの方は何をしているか分からなくなる点でしょうか。
