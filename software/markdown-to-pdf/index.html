node.js を利用した方法

Qiita: Linuxサーバ上でmarkdownのドキュメントをPDFに変換する

node の機能を使って実現しています。

pandoc, latex, inkscape の全部入り

Qiita: Markdownから素晴しいPDFをTravis CI上で作成することの考察

pandoc, latex, inkscape の入った docker を作成する方法が記載されています。 docker 使えるなら、これがいいかもしれません。

最終的なコマンドは下記のようになるそうです。

$ FILENAME="$BASE" pandoc -o "./target/$BASE.tex" -f markdown_github+footnotes+header_attributes-hard_line_breaks \
         --pdf-engine=lualatex --top-level-division=chapter --listings $FILTERS $f

また、利用している docker の Dockerfile が公開されています。

pandoc をほぼ基本設定で利用

PandocでMarkdownを段落番号のついたPDFに変換したい

こっちの方が、洗練されていない分、 pandoc の使い方として、わかりやすい気がします。 最終的なコマンドとしては、下記のようになるとのことです。 複数の .md ファイルが単一のディレクトリに含まれる状態を想定しています。 また、読み込まれる順番は、ファイル名順となるようです。

$ pandoc *.md -o output.pdf -V documentclass=ltjsarticle --latex-engine=lualatex -N --toc --highlight-style=tango
