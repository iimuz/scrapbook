BlenderでPythonスクリプトを利用する

2015-04-25 00:00:00 +0900 +0900 2020-06-01 17:08:46 +0900 +0900

BlenderでPythonを利用する。 ついでに、Blenderの物理シミュレーションをpythonで作成する。 作ったデータ: github: iimuz/blenderTutorials/cubePhysicsEngine

起動したBlenderからPythonスクリプトを利用する

情報処理I: 第15回を参考に以下のようにする。

物理シミュレーションをpythonで作成

株式会社CFlatの明後日スタイルのブログ: Blenderの物理シミュレーションをpythonスクリプトで作成 のスクリプトをいくつか変更して実行。

  • ループすると余計なオブジェクトが削除できなないので、ループして全部削除する。
- bpy.ops.object.delete()
+ for data in bpy.data.objects:
+     data.select = True
+     bpy.ops.object.delete()
  • 光源をスクリプトで追加すると、名称が変わっていたので、光源の呼び出しで利用する名称を変更す。
#照明
- bpy.data.objects["Lamp"].location = (0, 0, N+10)
- bpy.data.lamps["Lamp"].type = 'SUN'
+ bpy.ops.object.lamp_add();
+ bpy.data.objects["Point"].location = (0, 0, N+10)
+ bpy.data.lamps["Point"].type = 'SUN'
  • cameraをスクリプトで追加すると、sceneのcameraに設定されないので設定されるように変更。
    Blender beta: un blender Still rendering from command line with external Python script
# 動画作成

bpy.context.scene.camera = bpy.data.objects["Camera"]
bpy.context.scene.render.resolution_x = 400
bpy.context.scene.render.resolution_y = 300
bpy.context.scene.render.resolution_percentage = 100
bpy.context.scene.render.image_settings.file_format = 'AVI_JPEG'
  • ファイルの保存場所をblenderのファイルと同じ場所に変更。
- bpy.data.scenes["Scene"].render.filepath = "test.avi"
+ bpy.data.scenes["Scene"].render.filepath = "//test.avi"
