Gitのコマンド備忘録

2015-04-22 00:00:00 +0900 +0900 2020-07-20 07:57:18 +0900 +0900

gitのコマンドをよく忘れるので、利用するときに調べたらここに記載する。

コマンド

cloneする

git: 2.1 Gitの基本 - Gitリポジトリの取得を参考にすると、 カレントディレクトリにフォルダを自動で作成してcloneする場合は、以下の方法。

git clone <url>;

もし、cloneするディレクトリを指定する場合は、以下の方法でhogeフォルダに作成できる。

git clone <url> ./hoge;

さらにブランチも指定したいならば、 Qiita: リモートから特定のブランチを指定してcloneする を参考にすると、以下の方法で可能。

git clone -b <ブランチ名> <url>;

urlは、githubなら、対象リポジトリまで行って、 右のサイドバーにあるHTTPS clone URLというところをポチッすれば URLがクリップボードにコピーされる。

branchを作成する

サルでもわかるGit入門: 1. ブランチを作成するを参考に。

git branch <ブランチ名>;

pushする

初回のみ以下の方法で紐付が必要。

git push -u origin <ブランチ名>;

2回目以降は、以下でできる。

git push;

mergeする

git merge <ブランチ名>;

ステージにaddしたデータの削除

hogehoge foobar Blog Style5: git addでステージングしたファイルをアンステージング(キャンセル)する を参考に、キャッシュのみ削除する場合は以下。

git rm --cache ./hoge.txt

ファイルごと削除する場合は以下。

git rm ./hoge.txt

サブモジュールの追加

サブモジュールとして取り込む。

git submodule https://github.com/hoge hoge

リモートのアドレスを調べる

git remote -v
