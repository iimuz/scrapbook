静岡Developers勉強会: UnityとBlenderハンズオン 第1章 でUnityとBlenderの使い方を勉強する。

Unityでテトリスもどきを作成

noobtuts: Unity 2D Tetris Tutorialを参考に、 テトリスもどきを作成するようなので読みながら作ってみる。

テトリスの著作権については、ここを参照。

テトリスもどき: GitHub: likeTetris2d

Project Setup

当然のことながら2Dプロジェクト

Creating the Game Art

そのままの画像では面白くないので、BEIZ Graphicsから画像をダウンロード。

プロジェクトのAssetsフォルダに画像を直接追加するとUnityのGUI上で反映される。

ゲーム中のサイズはPixels to unit(Unity5だとPixels per unit?)になるらしい。

Note: the Pixels to Units property specifies the size in game.

Creating the Groups

ブロックの集合は、Prefubで作成する。

The Spawner

スポナーは、空オブジェクトで作成する。

Quanternion.indentityは、回転方向をデフォルトにする。

Note: transform.position is the Spawner’s position, Quaternion.identity is the default rotation.

The Grid Script

Math関数を利用するので、Systemを追加する。

Math.Roundがdoubleを返すようなので、floatにキャストする。

The Group Script

途中でlastFall変数が出てくるが、これはfloatでメンバ変数に定義する必要がある。

Update()の中が同じコードの繰り返しのため、適当にメンバ関数に切り分けて記述したほうが見やすい気がする。

他のオブジェクトの関数を呼び出す場合には以下のように書く。

FindObjectOfType<Spawner>().spawnNext();  // Spawn next grouup.

Blender編

サイコロを作成する。

サイコロ: GitHub: chapter01

設定情報の変更

FPSを29.97に設定する。

インターフェースを日本語化する。

  • User Preferences -> System -> International Fonts を Japanese にする。
    • TranslateのInterfaceとTooltipsをONにする。

選択を中心に回転およびPythonツールチップを表示

  • User Preferences -> Interface
    • 選択を中心に回転をON
    • Pythonツールチップを表示をOFF

3Dビュー Dynamic Spacebar Menu

  • User Preferences -> アドオン -> 3Dビュー Dynamic Space bar Menu をON

履歴操作

  • 元に戻す(Ctrl + Z)
  • やり直す(Shift Ctrl + Z)

サイコロ作成(UVマッピング)

動画(Youtube: Blender入門講座｜実践編｜UVマッピング)を参考にサイコロを作成する。 画像は、GitHub: yaju/ShizuokaDev_UnityAndBlenderからコピーしてくる。

表示モードは、UVマッピング用のUV Edittingを選択する。

オブジェクトのエディットモードで、 線選択で展開するときに切るところを選択し、 左側のシェーディング/UVタブからシームを付ける(Mark Seem)を選択する。

全選択で、シェーディング/UVタブから展開(Unwrap)を選択する。

貼り付ける画像(dice.png)を下側の画像->画像を開くから選択する。

3Dビューのシェーディングでテクスチャを選択すると、 画面上にテクスチャが張り付いて表示される。

頂点選択してpボタンで、頂点のピン止めを行うことで、移動しない点を作ることができる。

直方体のテクスチャで画像を設定する。 マッピングの座標をUVに設定する。 ただし、プレビューは正しく表示されないが、F12でレンダリングすればきちんと表示される。

適当に光源を移動して、表示している面が見えるようにする。

Unity用の修正

UnityやWebGLでは四角形のポリゴンを表示できないため、三角形ポリゴンに分割が必要らしい。

Cubeを全選択後に、Ctrl+Fから三角形に分割を選択する。

Jason形式にエクスポート

GitHub: mrdoob/three.jsからリポジトリをダウンロードして、zipファイルを解凍する。 解凍したフォルダ内にあるio_threeをBlenderのaddonにコピーする。

cp three.js-master/three.js-master/utils/exporters/blender/addons/io_three blender/version/scripts/addons/

Blenderで有効にするために、 User Preferences -> アドオン -> カテゴリ -> インポート/エクスポートから、 Three.js Formatを探してONにする。 表示されていなければ、更新ボタンを押せば出てくるはず。

ファイル -> エクスポート -> Three(json)でエクスポートする。
