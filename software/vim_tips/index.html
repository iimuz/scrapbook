Vim Tips

2019-04-13 11:08:37 +0900 +0900 2020-01-13 16:22:19 +0900 +0900

Normal mode

  • :e!: ファイルのリロード

Insert mode

  • Ctrl + j: 改行

Marks

  • :marks: mark の一覧1

Terminal

  • :term: ターミナルの起動2
    • :vert term: 垂直分割して開く
    • :bo term: 水平分割して画面最下部にウィンドウを開く
    • :top term: 水平分割して画面最上部にウィンドウを開く
  • :vert term ++close git log: git のログ確認2
  • :term ++close git blame %: 現在開いているファイルの blame 確認
