Tips

  • VSCode には Zen mode があり、アイコンなどを極力抑えた表示が可能です1。 Zen mode で Ctrl + Shift + P を検索すれば有効化できます。

ショートカット

プラグインで行うショートカットは記載していません。 特に vim plugin で実現できるところは、オリジナルのショートカットは書いていないです。

移動

  • タブの移動: Ctrl + Tab
  • ペイン切り替えまたは新規に作成: Ctrl + 数字
  • 折り返しの有無を切り替え: Alt + z
  • カーソル位置はそのままでスクロール: Ctrl + ↑ or ↓
  • カーソル位置はそのままでスクロール: Ctrl + PageUP or PageDown
  • 対応する括弧へ移動: Ctrl + Shift + \
    • windows の場合、右上の円マークは反応するが、右下のバックスラッシュは反応しない。
  • エディタを分割: Ctrl + \
    • windows では右上の円マークでしか反応しない。
  • アクティブなエディタのペインを移動する
    • Ctrl + Alt + ← or →
    • Shift + Alt + 数字
  • エディタグループの縦方向分割と横方向分割の切り替え: Alt + Shift + 0
  • インデントの開閉
    • カーソルのあるインデントを折りたたむ: Ctrl + Shift + [
    • カーソルのあるインデントを開く: Ctrl + Shift + ]
    • カーソル移動で折りたたんだところを超えようとすると勝手に開くので使いにくい。
  • フォントサイズの拡大と縮小: Ctrl + Shift + ; or -
  • 全参照検索して移動: Ctrl + Shift + o
  • シンボルの定義を表示: Ctrl + マウスホバー
  • 参照または定義の結果をインライン表示: Shift + F12 or Alt + F12
  • シンボルのリネーム: F2

ファイル編集

  • 補完: Ctrl + Space
  • 選択行を上または下へ移動: Alt + ↑ or ↓
  • 選択行を上または下へコピー: Alt + ↑ or ↓

検索 or 置換

  • ファイル検索: Ctrl + P
  • 全体
    • 検索: Ctrl + Shift + f
    • 置換: Ctrl + Shift + h
    • 検索結果の移動: F4 or Shift + F4
  • 検索ボックス内で履歴表示: Alt + ↑ or ↓

何かを開く

  • 設定画面を開く: Ctrl + ,
  • エクスプローラを開く: Ctrl + Shift + e
  • 検索を開く: Ctrl + Shift + f
  • ソース管理を開く: Ctrl + Shift + g
    • gitlens プラグインを導入している場合は、 キーバインドが重複するため Ctrl + Shift + g g に変更されている。
  • デバッグを開く: Ctrl + Shift + d
  • 拡張機能を開く: Ctrl + Shift + x
  • 出力を開く: Ctrl + Shift + u
  • デバッグコンソールを開く: Ctrl + Shift + y
  • ターミナルを開く: Ctrl + @
  • 問題(Problems)を開く: Ctrl + Shift + m
    • 次の問題へフォーカスを移動する: Alt + F8
    • quickfix を開く: Ctrl + .

vim plugin

  • GitHub VSCodeVim/Vim
  • サイドバー
    • サイドバーへの移動: Ctrl + w, h
    • サイドバーからメイン部分への移動: Ctrl + w, w
    • サイドバーを開く: Ctrl + n
    • サイドバーでフォルダを開く: o
    • サイドバーでカーソルのあるファイルを開く: Enter
  • ページ送り: Ctrl + f or Ctrl + b
  • 保存: :w
  • ファイル内検索:/
    • 次の検索結果: n
    • 前の検索結果: N
  • 折り畳み
    • 一つ分折り畳み/展開: z c, z o
    • 折り畳みの開閉/再帰的開閉: z a, z A
    • バッファ内の全ての折り畳む/開く: z M, z R
  • easy motion
    • 一文字検索: <leader><leader> s <char>
      • f だと前方検索、 F だと後方検索
      • t だと前方検索語に手前にカーソルをセット、 T は後方絵検索で手前にカーソルセット
    • 前方単語先頭にカーソルセット: <leader><leader> w
      • b, e, ge だとそれぞれ、後方単語先頭、前方単語末尾、後方単語末尾。
      • j, k はそれぞれ、前方行頭もしくは後方行頭
    • 前方絞込検索: <leader><leader> /
  • surround
    • カーソル下のテキストを囲むオブジェクトを削除: d s <削除するオブジェクト>
    • カーソル下のテキストを囲むオブジェクトを変更: c s <変更前のオブジェクト> <変更後のオブジェクト>
    • 選択範囲を囲む: S <囲むオブジェクト>
  • commentary
    • ラインコメント: g c <CR>
    • ブロックコメント: g C

調べること

  • Problems のヒントを挿入形式で出力する方法が不明です。 Alt + F8でフォーカスすると表示されるものを、カーソル下の問題に対して開いてほしいです。
